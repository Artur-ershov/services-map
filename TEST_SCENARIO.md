# Сценарий тестирования мобильной версии карты сервисов

## Подготовка
1. Открыть приложение в браузере
2. Установить размер окна браузера: 375x667 (iPhone SE) или использовать DevTools mobile emulation
3. Убедиться, что сервер запущен на порту 8000

## Тест 1: Базовая загрузка и отображение
**Цель**: Проверить, что все основные элементы отображаются корректно

**Шаги**:
1. Открыть `http://localhost:8000/index.html`
2. Проверить видимость элементов:
   - [ ] Кнопка выбора этажа видна и кликабельна
   - [ ] Поле поиска видно и доступно
   - [ ] Фильтры категорий видны и прокручиваются горизонтально
   - [ ] Карта отображается (высота ~400px)
   - [ ] Список сервисов виден и прокручивается вертикально

**Ожидаемый результат**: Все элементы видны, ничего не обрезано, нет горизонтального скролла страницы

---

## Тест 2: Фильтры категорий
**Цель**: Проверить работу фильтров на мобильных устройствах

**Шаги**:
1. Проверить горизонтальную прокрутку фильтров:
   - [ ] Все кнопки фильтров видны (можно прокрутить)
   - [ ] Кнопки не переносятся на новую строку
   - [ ] Прокрутка плавная (touch scrolling)
2. Кликнуть на кнопку "Переговорные"
   - [ ] Фильтр активируется
   - [ ] Список обновляется
   - [ ] Подфильтры появляются (если есть данные)
3. Проверить подфильтры:
   - [ ] Подфильтры прокручиваются горизонтально
   - [ ] Можно выбрать подфильтр
   - [ ] Список фильтруется корректно

**Ожидаемый результат**: Фильтры работают, прокручиваются, не обрезаются

---

## Тест 3: Поиск
**Цель**: Проверить работу поиска на мобильных

**Шаги**:
1. Кликнуть на поле поиска
   - [ ] Клавиатура открывается
   - [ ] Поле доступно для ввода
2. Ввести текст "кофе"
   - [ ] Список фильтруется в реальном времени
   - [ ] Результаты отображаются
3. Очистить поиск
   - [ ] Список возвращается к исходному состоянию

**Ожидаемый результат**: Поиск работает корректно, клавиатура не перекрывает интерфейс

---

## Тест 4: Выбор этажа
**Цель**: Проверить работу выбора этажа

**Шаги**:
1. Кликнуть на кнопку "Выбрать этаж"
   - [ ] Открывается модальное окно выбора этажа
   - [ ] Окно помещается на экран
   - [ ] Можно прокрутить список этажей
2. Выбрать другой этаж
   - [ ] Карта обновляется
   - [ ] Список обновляется
   - [ ] Модальное окно закрывается
3. Проверить кнопки навигации по этажам (▲ ▼)
   - [ ] Кнопки видны
   - [ ] Работают корректно (▲ - выше, ▼ - ниже)

**Ожидаемый результат**: Выбор этажа работает, модальное окно не выходит за границы экрана

---

## Тест 5: Список сервисов
**Цель**: Проверить работу списка сервисов

**Шаги**:
1. Проверить прокрутку списка:
   - [ ] Список прокручивается вертикально
   - [ ] Прокрутка плавная (touch scrolling)
   - [ ] Все элементы списка видны
2. Проверить элементы списка:
   - [ ] Индикатор категории отображается
   - [ ] Название сервиса видно полностью
   - [ ] Описание/атрибуты видны
3. Проверить выделение активного элемента:
   - [ ] При клике элемент выделяется
   - [ ] Элемент прокручивается в видимую область

**Ожидаемый результат**: Список работает корректно, все элементы доступны

---

## Тест 6: Карта
**Цель**: Проверить работу карты на мобильных

**Шаги**:
1. Проверить отображение карты:
   - [ ] Карта отображается (высота ~400px)
   - [ ] Области сервисов видны
   - [ ] Карта не выходит за границы экрана
2. Проверить взаимодействие с картой:
   - [ ] Можно тапнуть на область сервиса
   - [ ] Область подсвечивается при тапе
   - [ ] Открывается попап с информацией
3. Проверить масштабирование (если реализовано):
   - [ ] Можно увеличить/уменьшить карту
   - [ ] Можно перемещать карту

**Ожидаемый результат**: Карта работает, области кликабельны

---

## Тест 7: Попап с информацией
**Цель**: Проверить отображение попапа на мобильных

**Шаги**:
1. Открыть попап (клик на элемент списка или области карты)
   - [ ] Попап открывается
   - [ ] Попап помещается на экран (не выходит за границы)
   - [ ] Можно прокрутить содержимое попапа
2. Проверить элементы попапа:
   - [ ] Кнопка закрытия (×) видна и работает
   - [ ] Галерея изображений (если есть) отображается
   - [ ] Название сервиса видно
   - [ ] Описание видно
   - [ ] Контакты видны
   - [ ] Кнопка "Подробнее" видна (если есть ссылка)
3. Проверить галерею (если есть несколько изображений):
   - [ ] Изображения отображаются
   - [ ] Можно свайпнуть влево/вправо
   - [ ] Индикаторы показывают текущее изображение
   - [ ] Кнопки навигации работают
4. Закрыть попап:
   - [ ] Клик на × закрывает попап
   - [ ] Клик вне попапа закрывает его (если реализовано)

**Ожидаемый результат**: Попап корректно отображается, все элементы доступны, можно прокрутить

- Samsung Galaxy S20: 360x800

**Шаги**:

   - [ ] Нет горизонтального скролла страницы
   - [ ] Попап помещается на экран
   - [ ] Фильтры прокручиваются
   - [ ] Список прокручивается

**Ожидаемый результат**: Приложение работает на всех размерах экранов
